Restaurant Manager

Narzędzie do zarządzania restauracją z funkcjami obsługi menu, rezerwacji i ocen klientów.

Opis projektu

Restaurant Manager to aplikacja do zarządzania restauracją, oferująca szeroki zakres funkcjonalności usprawniających pracę restauracji oraz interakcję z klientami. Projekt składa się z dwóch głównych części:

  • Backend Napisany w Node.js z użyciem Express.js i MongoDB (online). Odpowiada za rejestrację i logowanie użytkowników (JWT), zarządzanie restauracjami, system ocen i recenzji oraz system rezerwacji stolików. Wykorzystuje bibliotekę Mongoose do operacji na danych i bcrypt.js do szyfrowania haseł.
  • Frontend Zrealizowany w Angularze z wykorzystaniem Angular Material i Bootstrapa. Zapewnia intuicyjny interfejs użytkownika, atrakcyjny wygląd i przejrzystą nawigację. Umożliwia dynamiczne filtrowanie restauracji oraz przeglądanie szczegółowych informacji.

Główne funkcjonalności:

  • 📝 System rezerwacji stolikówRezerwacje z możliwością edycji lub anulowania.
  • ⭐ System ocen i recenzji Użytkownicy mogą oceniać restauracje i dodawać komentarze.
  • 🔒 System logowania i rejestracji Obsługuje JWT, z dodatkowymi polami opcjonalnymi.
  • 📋 Zarządzanie menu restauracji Widok interaktywnego menu dostępny w szczegółach restauracji.
  • 📈 System rekomendacji Proponuje trzy restauracje bazując na ocenach innych użytkowników oraz preferencjach (algorytm k-NN).
  • 👤 Panel użytkownika Pozwala na edycję profilu, zmianę awatara oraz podgląd i zarządzanie rezerwacjami.
  • 📂 Rozsuwane menu boczneUmożliwia szybki dostęp do kluczowych funkcji aplikacji.
  • 🔍 Zaawansowane filtrowanie restauracjiOpcja wyszukiwania restauracji według lokalizacji, kuchni oraz zakresu cenowego.

Technologie

Node.js Express.js MongoDB Angular Angular Material Bootstrap JWT (JSON Web Token) Mongoose bcrypt.js dotenv CORS

Rozwój projektu

📝 System ocen i recenzji

Rozbudowanie istniejącego systemu ocen o dodatkowe funkcje, takie jak oznaczanie pomocnych recenzji czy filtrowanie komentarzy.

💳 Integracja z systemami płatności

Dodanie bezpiecznych metod płatności online, np. Stripe lub PayPal, aby umożliwić użytkownikom szybkie opłacanie zamówień.

📊 Rozbudowana analityka sprzedaży

Implementacja panelu analitycznego dla właścicieli restauracji, zawierającego raporty dotyczące zamówień, popularnych dań oraz statystyk przychodów.

🍔 System zamówień z restauracji

Umożliwienie użytkownikom składania zamówień na wynos bezpośrednio z poziomu aplikacji.

🔔 System powiadomień

Dodanie powiadomień push oraz e-mailowych dotyczących statusu rezerwacji, promocji czy nowości w menu.

🔍 Zaawansowane filtrowanie restauracji

Rozszerzenie obecnych filtrów o dodatkowe kryteria, takie jak godziny otwarcia, dostępność miejsc czy promocje.
