Event Manager

System do zarządzania wydarzeniami z możliwością rejestracji, edycji i zarządzania uczestnikami.

ASP.NET Core Entity Framework Core Microsoft Identity SQL Server .NET MVC

Opis projektu

Event Manager to nowoczesna aplikacja webowa stworzona w technologii ASP.NET Core, umożliwiająca kompleksowe zarządzanie wydarzeniami. System obsługuje cztery główne role: Administratora, Organizatora, Moderatora i Użytkownika. Każdy z nich posiada dedykowane funkcjonalności wspierające zarządzanie wydarzeniami oraz uczestnikami.

Aplikacja wykorzystuje bazę danych SQL Server LocalDB oraz wzorzec projektowy MVC, który zapewnia przejrzystą strukturę kodu oraz łatwość w rozbudowie systemu.

Administrator

Administrator pełni kluczową rolę w systemie. Posiada pełny dostęp do zarządzania wydarzeniami, użytkownikami i statystykami. Może edytować, usuwać oraz tworzyć nowe wydarzenia, a także zmieniać role użytkowników oraz zarządzać zapisami.

Organizator

Organizator ma możliwość tworzenia nowych wydarzeń oraz edycji i zarządzania tylko tymi wydarzeniami, które sam utworzył. System w przyszłości umożliwi także podgląd listy uczestników oraz generowanie raportów.

Moderator

Moderator pełni funkcję wspierającą. Aktualnie może przeglądać wydarzenia, a w przyszłości będzie odpowiedzialny za moderację treści i zarządzanie zgłoszeniami uczestników.

Użytkownik

Użytkownik może przeglądać dostępne wydarzenia, zapisywać się na nie oraz zarządzać swoimi zgłoszeniami. Posiada dedykowany panel z listą nadchodzących wydarzeń oraz możliwością rezygnacji z uczestnictwa.

👤 Użytkownik

  • Rejestracja i logowanie z przypisaniem ról
  • Przeglądanie listy dostępnych wydarzeń
  • Wyświetlanie szczegółowych informacji o wydarzeniach
  • Zapisywanie się na wydarzenia z możliwością rezygnacji
  • Podgląd własnych zapisanych wydarzeń w dedykowanym panelu

🛡️ Administrator

  • Dodawanie, edytowanie i usuwanie wydarzeń
  • Zmiana ról użytkowników i zarządzanie kontami
  • Podgląd statystyk systemu
  • Zarządzanie zapisami na wydarzenia

📋 Organizator

  • Tworzenie nowych wydarzeń
  • Edycja i zarządzanie wydarzeniami, które sam utworzył
  • Podgląd listy uczestników swoich wydarzeń

🛑 Moderator

  • Przeglądanie wydarzeń
  • Moderowanie treści oraz zarządzanie zgłoszeniami uczestników
